Each action can be called in your code using standard WordPress way, using `do_action()`_ function:

.. _`do_action()`: https://codex.wordpress.org/Function_Reference/do_action

.. code-block:: php

    do_action( $action_tag, $arg1, $arg2 ... );


To hook on action, use `add_action()`_ WP function:

.. _`add_action()`: https://codex.wordpress.org/Function_Reference/add_action

.. code-block:: php

    add_action( $action_tag, $callback, $priority = 10, $accepted_args = 1 );
