Each filter can be applied in your code using standard WordPress way, using `apply_filters()`_ function:

.. _`apply_filters()`: https://codex.wordpress.org/Function_Reference/apply_filters

.. code-block:: php

    apply_filters( $filter_tag, $arg1, $arg2 ... );


To hook on filter, use `add_filter()`_ WP function:

.. _`add_filter()`: https://codex.wordpress.org/Function_Reference/add_filter

.. code-block:: php

    add_filter( $filter_tag, $callback, $priority = 10, $accepted_args = 1 );
